How to Code — A short, brilliant journey

Why learn coding?

Code is a superpower — it turns ideas into things people can use. From websites to robots and art, code is the tool creators use to make change.

  • Usefulness: Build tools, automate tasks, create games.
  • Thinking: Coding trains clear logical thinking — a skill for life.
  • Opportunity: Jobs, research, startups — doors open when you can build.

Think: what would you create if you could code? Keep that idea — we'll help you build it step by step.

HTML — structure & meaning

HTML is the skeleton of a web page. It tells the browser what each piece of content *is*: a heading, a paragraph, a list, a link, an image.

<h1>Hello</h1>
<p>This is a paragraph.</p>
<ul>
  <li>Item A</li>
</ul>

Try: open a plain text file, paste the snippet, save as example.html, and open it in your browser — that's instant magic.

CSS — style and layout

CSS makes HTML look beautiful. Colors, spacing, animations, and responsive layouts all come from CSS.

body { font-family: system-ui; background:#fff; color:#111; }
h1 { color: #2b6cb0; }

Play: add a <link rel="stylesheet" href="styles.css"> to your HTML and change colors live.

JavaScript — behavior and logic

JS lets pages respond: clicking buttons, fetching data, animating things — it's how you make sites interactive.

const btn = document.querySelector('button');
btn.addEventListener('click', () => alert('Hello!'));

Practice: add a script to your HTML, log something to the console, and watch the browser respond.

Final challenge: Pick one thing: make a small page that shows your name and changes color when clicked. Build it — you can do it.
